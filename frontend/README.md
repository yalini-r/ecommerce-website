# E-Commerce Website (Frontend)

## Project Overview
This E-Commerce Website is a frontend web application developed to simulate an online shopping experience. Users can browse products, filter by categories, add items to the cart, update quantities, and view the total price dynamically.

This project focuses on frontend development using React and does not include backend or payment gateway integration.

---

## Tech Stack
- HTML
- CSS
- JavaScript
- React
- Git & GitHub

---

## Features
- Category-based product listing (Men, Women, Kids)
- Product search functionality
- Add to cart and remove from cart
- Quantity increment and decrement
- Automatic total price calculation
- Responsive design
- Reusable React components
